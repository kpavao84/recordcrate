* Recordcrate

Keep track of your records.

A work in progress.
** Getting Started
Clone the repo:
#+BEGIN_SRC sh
  git clone https://github.com/kpavao84/recordcrate.git
#+END_SRC

Set up Python:
#+BEGIN_SRC sh
  cd recordcrate/api/
  virtualenv env --no-site-packages
  source env/bin/activate
  pip install -r requirements.txt
#+END_SRC

Run the server:
#+BEGIN_SRC sh
  cd vinyl/
  python manage.py runserver
#+END_SRC
